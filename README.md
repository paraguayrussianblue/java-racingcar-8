# java-racingcar-precourse
----
## 구현 기능 목록

### 1) 입력 받기
- 자동차 이름 목록을 입력 받는다.

- 시도할 횟수를 입력 받는다.


### 2) 게임 진행

- 입력 받은 자동차 이름이 다섯 글자를 초과하거나, 이름이 빈 경우 IllegalArgumentException을 발생시킨다.
- 입력 받은 시도 횟수가 정수가 아니거나 음수인 경우 IllegalArgumentException을 발생시킨다.
- 각 자동차의 초기 위치는 0이다.
- 매회차에 각 자동차마다 0~9 사이의 무작위의 숫자를 뽑는다.
- 뽑은 숫자가 4 이상일 경우 해당 자동차가 1칸 전진한다.

### 3) 회차 및 게임 결과 출력

- 매회차가 진행된 후 각 자동차의 이름과 위치를 출력한다.
- 모든 회차가 진행되고 나면, 위치가 출발점으로부터 가장 먼 자동차들의 이름을 최종 우승자로 출력한다.
